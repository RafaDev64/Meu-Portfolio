Meu E-commerce – Projeto Pessoal

Esse projeto foi um grande aprendizado para mim! Desenvolvi uma loja virtual completa, com um visual moderno e responsivo, pensando tanto na experiência do cliente quanto na facilidade de administração. Implementei um sistema de login onde é possível escolher entre entrar como Administrador ou Usuário. O administrador tem acesso a um painel exclusivo para cadastrar novos produtos, enquanto os usuários podem criar suas próprias contas, informando nome, senha, CPF, e-mail e telefone. O site permite buscar, filtrar e adicionar produtos ao carrinho, com feedback visual para cada ação. O carrinho é dinâmico e fácil de usar, e todo o layout foi pensado para ser agradável em qualquer dispositivo. Utilizei HTML, CSS e JavaScript puro, com armazenamento local dos dados (localStorage), validação de formulários e boas práticas de usabilidade. Esse projeto representa bem minha evolução em front-end e minha preocupação com a experiência do usuário!

Dashboard de Finanças Pessoais

Este projeto é um painel simples e intuitivo para controle financeiro pessoal. Com ele, é possível registrar receitas e despesas, visualizar o saldo atualizado em tempo real e acompanhar todas as movimentações em uma tabela organizada. Funcionalidades:

• Cadastro rápido de receitas e despesas.
• Cálculo automático do saldo, total de receitas e despesas.
• Visualização das transações em tabela, com opção de remoção.
• Interface responsiva, moderna e de fácil navegação.
• Armazenamento local dos dados (localStorage), mantendo as informações salvas mesmo após fechar o navegador.
• Botão de saída com confirmação centralizada para maior segurança.
Ideal para quem busca praticidade no acompanhamento das finanças do dia a dia!.
